# solar-selector-mvp_v3

静的サイト（HTML/CSS/JS）だけで動く、太陽光＋蓄電池の「15年評価→推奨案抽出」ツールです。

## 起動方法
JSONをfetchするので、**file://** 直開きではなくローカルサーバーで起動してください。

### 例1）Python
```bash
cd solar-selector-mvp_v3
python -m http.server 8000
```
→ ブラウザで `http://localhost:8000/solar-selector-mvp_v3/index.html`

### 例2）VSCode
拡張機能 **Live Server** で `index.html` を開く。

## 仕様（今回反映）
- PV探索範囲：**1.00kW〜13.50kW / 0.01刻み**
- 純利益（15年）：`15年便益（自家消費 + 売電） - 概算総額（中央値）`
- 概算総額（中央値）：`(287000 + 175500x + 200000z - 1200x^2 - 3000z^2) * (1 + 利益率)`
- 概算レンジ：`中央値 ± (x+z)*25000`

## 工務店設定
右下の⚙ または **Shift+9** で開く  
パスコード初期値：`ogw`（設定画面で変更可）

- 利益率（%） … 結果画面には表示されません（内部計算のみ）
- 固定費（円/月）
- 未入力時の単価（円/kWh）
- 夜間比率（%）
- 発電量（kWh/kW/日）
- PV探索範囲（kW）
- “目劣り”判定（許容差）
- デバッグ表示（工務店向けの内訳表示）
